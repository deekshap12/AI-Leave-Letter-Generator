# AI Leave Letter Generator

## Project Description
This project is a Generative AI application that generates formal leave letters automatically using the Gemini API. The user provides details such as student name, class, reason for leave, and leave duration. The system processes the input and generates a properly formatted leave letter.

## Technologies Used
- Python
- Google Colab
- Gemini API
- Google AI Studio

## How the Application Works
1. User enters leave details.
2. The program creates a structured prompt.
3. The prompt is sent to the Gemini AI model.
4. The AI generates a formal leave letter.
5. The output is displayed to the user.

## Example Output
The application generates a professional leave letter based on the user input.
